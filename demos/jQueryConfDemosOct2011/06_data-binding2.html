JsViews: Data binding - compact linking API syntax

HTML:


...
    <td data-link="firstName" />
    <td>
        <input data-link="firstName" />
    </td>
...

Script:

// Compile template
$.templates( "personTmpl", "#personTmpl" );

// Render and link (compact syntax): data-link details
// container element to people, using the personTmpl template
$.link.personTmpl( "#details", people );
