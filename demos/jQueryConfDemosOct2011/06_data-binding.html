JsViews: Data binding

HTML:


...
    <td data-link="firstName" />
    <td>
        <input data-link="firstName" />
    </td>
...

Script:

// Compile template
$.templates( "personTmpl", "#personTmpl" );

// Render to string, and insert as HTML
$( "#details" ).html(
    $.render.personTmpl( people )
);

// ... and data-link to people
$.link( "#details", people );

Other possible data-binding paths:

<input data-link="{convert:manager.firstName:convertBack} title{:manager.firstName + manager.lastName}" ... />
