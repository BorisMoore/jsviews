JsRender: Using {{if}} and {{else}} to render conditional sections

TitleLanguages

HTML:

<td>
    {{if languages}}
        ...
    {{else subtitles}}
        ....
    {{else}}
        ...
    {{/if}}
</td>

Data:

var movies = [
    {
        title: "Meet Joe Black",
        languages: "English and French",
        subtitles: "English"
    },

        title: "Eyes Wide Shut",
        subtitles: "French and Spanish"
    },
    ...
];
