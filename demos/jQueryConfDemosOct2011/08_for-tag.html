JsRender: Using {{for}} to render hierarchical data - inline nested template

TitleLanguages

Data:

var movie = {
    title: "Eyes Wide Shut",
    languages: [
        { name: "French" },
        { name: "Mandarin" },
        { name: "Spanish" }
    ]
}

HTML:

<td>{{:title}}</td>
<td>
    {{for languages}}
        <div>{{:name}}</div>
    {{/for}}
</td>
